# slack-ninja
開發Slack APP, 實作以下機制:
- Incoming Webhooks
- Interactive Components
- Slash Commands
- Event Subscriptions
- Bots

使用之Slack Web API:
- channels.history
- chat.command
- chat.delete
- chat.postMessage
- chat.postEphemeral
- dialog.open

提供功能如下:
- 電商平台點數查詢
- [抓取PTT NBA BOX][nba]
- Heroku Usage查詢
- 百米家新商品通知
- 捷運票價及乘車時間
- 查詢可用任務
- 刪除特定日期訊息
- [台北市天氣查詢][weather]
- 台北市公車動態查詢
- 台北市公車路線查詢
- [高鐵時刻表與票價查詢][thsr]
- [各縣市空氣品質概況][aqi]
- 雙北上映影片查詢
- [雙北上映影片時刻表][movie]
